# Mtskheta Tourism Website

## Description
This project is a responsive tourism website about Mtskheta, one of the oldest cities in Georgia and a UNESCO World Heritage Site.

The website provides information about Mtskheta's history, famous landmarks, culture, and tourist attractions.

## Features
- Responsive design for different screen sizes
- Navigation menu
- Information about historical places
- Beautiful images and sections
- Modern layout using Flexbox

## Technologies Used
- HTML5
- CSS3
- Flexbox
- Responsive Web Design
